# Mobile Responsiveness Checklist

## Layout
- [ ] Single column on mobile
- [ ] Proper spacing and padding
- [ ] Touch targets minimum 44px
- [ ] No horizontal scrolling

## Forms
- [ ] Full-width inputs on mobile
- [ ] Proper input types (tel, email, number)
- [ ] Virtual keyboard appropriate

## Navigation
- [ ] Bottom navigation on mobile
- [ ] Hamburger menu working
- [ ] Back navigation clear

## Specific Pages
- [ ] Dashboard cards stack
- [ ] Invoice list cards full width
- [ ] Invoice editor scrollable
- [ ] Line items table responsive
- [ ] Photo grid adapts

## Testing Devices
- [ ] iPhone SE (small - 375px)
- [ ] iPhone 14 (medium - 390px)
- [ ] iPhone 14 Pro Max (large - 430px)
- [ ] iPad (tablet - 768px)
- [ ] Android devices
